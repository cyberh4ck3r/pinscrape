# pinscrape

a discord bot with slash commands to scrape pinterest images directly in discord.

## usage

1. clone the repo
2. put your discord bot token in `.env` file (get one at [discord.com/developers/applications](https://discord.com/developers/applications))
3. run `pip install -r requirements.txt`
4. run `python bot.py`
5. use `/pinscrape` to search for images
6. use `/autopost` to automatically post images in a channel
